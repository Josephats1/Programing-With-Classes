Abstraction

Definition: Abstraction is the concept of hiding the complex implementation details and exposing only the necessary parts of a system.

Usage in My Program:

In my Exercise Tracking Program, I created an abstract Activity class with methods like GetDistance() and GetSpeed(). These methods were not implemented in the base class but were meant to be overridden by derived classes.

Flexibility for Future Changes:

This design allows easy extension of the program to add new activities in the future without modifying the core structure of the program.

Encapsulation

Definition: Encapsulation is the bundling of data and methods that operate on the data, restricting direct access to some of the object's components.

Usage in My Program:

In the Activity class, I made member variables like distance, speed, and laps private and provided public methods to access or modify them safely.

Flexibility for Future Changes:

This allows for controlled changes to the internal workings of an object without impacting other parts of the program.

Inheritance

Definition: Inheritance enables a class to inherit properties and methods from another class, facilitating code reusability and creating a hierarchical structure.

Usage in My Program:

The Activity class is the base class, and Running, Cycling, and Swimming classes inherit from it to reuse common functionality and add specific behavior.

Flexibility for Future Changes:

New activity types can be easily added by creating new subclasses that inherit from Activity, reusing common logic.

Polymorphism

Definition: Polymorphism allows different classes to be treated as instances of a common superclass, enabling method overriding and dynamic method invocation.

Usage in My Program:

The Activity class defines abstract methods for distance and speed. These methods are overridden in each derived class like Running, Cycling, and Swimming to provide specific implementations.

Flexibility for Future Changes:

Polymorphism allows new activities to be introduced without altering the existing code, ensuring scalability.
